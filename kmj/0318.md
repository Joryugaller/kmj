# 세미콜론 과제

1. git은 로컬 저장소의 모든 변경을 기록하는 버전관리 소프트웨어, GitHub는 git을 사용하는 프로젝트를 지원하는 웹 호스팅 서비스
2. 마크다운을 써야하는 이유는 문서 구조를 간단히 구성하고, 마크다운을 지원하는 플랫폼이 많고, 버전 관리가 용이하기 때문이다. 주로 사용되는 분야는 README 작성, 깃허브, 노션, 블로그 등에 쓰인다.
3. python:웹 서버, 어플리케이션, 딥러닝 등에 사용된다.
   C언어:자동화,GUI,응용 및 시스템 프로그램 등에 폭넓게 활용된다.
   JavaScript:웹 프론트엔드에서 사용자와 상호작용하기 위해 사용된다.
   Java:대규모 웹 애플리케이션, 안드로이드 앱 개발 등에 사용된다.
4. 라이브러리는 개발자들이 자주 사용하는 코드의 집합, 프레임워크는 개발자들이 애플리케이션을 개발하는데 사용되는 구조를 제공하는 것이고, API는 서로 다른 소프트웨어 애플리케이션들이 상호작용할 수 있도록 정의된 규칙의 집합이다.
